🗓️ ปฏิทินการดำเนินการ — การจัดทำแผนพัฒนาจังหวัดน่าน พ.ศ. 2571–2575

(ร่าง) ปฏิทินการประสานแผนระดับพื้นที่ · เป้าหมายการพัฒนาจังหวัดน่าน 20 ปี ฉบับปรับปรุง · แผนพัฒนาจังหวัดน่าน 2571–2575 และแผนปฏิบัติราชการประจำปีงบประมาณ 2571

📌หมายเหตุ
1. ฝ่ายเลขานุการ ก.น.บ. ประกอบด้วย สำนักงานสภาพัฒนาการเศรษฐกิจและสังคมแห่งชาติ สำนักงบประมาณ และกระทรวงมหาดไทย
2. กฎหมาย ระเบียบ นโยบาย หลักเกณฑ์ และแนวทางปฏิบัติที่เกี่ยวข้อง:
  • พระราชกฤษฎีกาการบริหารงานเชิงพื้นที่แบบบูรณาการ พ.ศ. 2565
  • ระเบียบกระทรวงมหาดไทยว่าด้วยการจัดทำแผนและประสานแผนพัฒนาพื้นที่ในระดับอำเภอและตำบล พ.ศ. 2562 (ฉบับปรับปรุง ปีงบประมาณ พ.ศ. 2568)
  • ระเบียบกระทรวงมหาดไทยว่าด้วยการจัดทำแผนพัฒนาท้องถิ่น พ.ศ. 2548 และที่แก้ไขเพิ่มเติม
  • แนวทางปฏิบัติการเตรียมความพร้อมเพื่อจัดทำแผนพัฒนาท้องถิ่น ฉบับที่ 3 (พ.ศ. 2571–2575)
  • หลักเกณฑ์และวิธีการจัดทำเป้าหมายการพัฒนาจังหวัด 20 ปี ฉบับปรับปรุง ตามมติ ก.น.บ. ครั้งที่ 1/2569 (23 เม.ย. 2569)
  • นโยบาย หลักเกณฑ์ และวิธีการจัดทำแผนของจังหวัดและกลุ่มจังหวัด ปีงบประมาณ พ.ศ. 2571–2575 ตามมติ ก.น.บ. ครั้งที่ 1/2569
  • หลักเกณฑ์การจัดทำแผนปฏิบัติราชการประจำปีของจังหวัดและกลุ่มจังหวัด ปีงบประมาณ พ.ศ. 2571–2575 ตามมติ ก.น.บ. ครั้งที่ 1/2569
3. อาจมีการปรับเปลี่ยนกำหนดเวลาตามความเหมาะสม

1️⃣ขั้นตอนที่ 1 — เปิด Google Sheets และสร้าง Apps Script
  1. เปิด Google Sheets (คลิกที่นี่) ↗
  2. คลิกเมนู Extensions → Apps Script
  3. ลบโค้ดเดิมทั้งหมด แล้ว วางโค้ดด้านล่าง ลงไป
  4. กด 💾 Save แล้วคลิก Deploy → New deployment
  5. เลือก Type: Web app, Execute as: Me, Who has access: Anyone (ทุกคน) ⚠ ต้องเลือก "ทุกคน" เท่านั้น
  6. คลิก Deploy แล้วคัดลอก Web App URL
  7. วาง URL ในช่องด้านล่าง กด 💾 บันทึก แล้ว 🧪 ทดสอบ — ถ้าขึ้น ✅ แสดงว่าพร้อมใช้งาน (หากขึ้น ⚠ สีเหลือง ระบบจะยังบันทึกข้อมูลได้ปกติด้วยวิธีสำรองอัตโนมัติ)
